# Извлечение email адресов из документов

Скрипт для автоматического поиска и извлечения адресов электронной почты из офисных документов и PDF файлов.

## Возможности

- ✅ Рекурсивный поиск файлов во всех подпапках
- ✅ Поддержка форматов: DOCX, DOC, XLSX, XLS, PPTX, PPT, PDF, TXT
- ✅ Графический интерфейс с выбором папки
- ✅ Отображение прогресса обработки
- ✅ Сохранение результатов в CSV и Excel
- ✅ Показ статуса обработки каждого файла

## Установка

1. Установите Python 3.7 или выше
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Запустите скрипт:

```bash
python find_emails.py
```

1. Нажмите кнопку "Выбрать папку"
2. Выберите папку с документами
3. Нажмите "Начать обработку"
4. Дождитесь завершения обработки
5. Результаты будут сохранены в выбранной папке:
   - `найденные_email.csv` - таблица в формате CSV
   - `найденные_email.xlsx` - таблица в формате Excel

## Формат результатов

Таблица содержит следующие столбцы:
- **Email адрес** - найденный адрес электронной почты
- **Количество файлов** - в скольких файлах найден этот email
- **Файлы** - список файлов, в которых найден email

## Примечания

- Для старых форматов .doc и .ppt может потребоваться дополнительная настройка
- Скрипт обрабатывает все файлы рекурсивно, включая вложенные папки
- Прогресс обработки отображается в реальном времени
